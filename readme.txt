== Description ==
Protect PDF files by passwords on your WordPress Website.
PDF Password Protect plugin offers an easy way to set passwords and control edit permissions to your PDF files.

== Features ==

* Add user password to prevent opening the PDF file.
* Add owner password to disabled PDF file edit permissions:

= Permissions to disable =
* copy: copy text from PDF file.
* print: print the PDF file.
* comment: add comments to PDF file.
* comment - fill form: Add comments and fill forms to PDF file.

== Requirements ==
The plugin needs two packages to be installed on your server in order to handle complex PDF files that have forms - annotations - bookmarks and to handle some PDF files which use Compressed Cross-reference and Object Streams for data structure.

* QPDF min V 9.1.1
* GhostScript

== How to use ==
* Activate the plugin
* Upload PDF file
* Access the PDF attachment edit page from media page
* You can add password to the PDF file from the Sidebar metabox "PDF password options".
